What Powers It

Real data.
Not scraped guesses.

Every alert, lead, and estimate is grounded in live public data and verified commercial APIs — not AI hallucinations or stale databases.

🏗️ Shovels API

Building permit data across 130M+ US properties. New filings, expired permits, contractor history, permit valuation.

Used by: RoofBot, PlumbBot, GardenBot

📋 LADBS / Socrata

Los Angeles Department of Building and Safety open data. Real-time permit filings, work descriptions, zip-code level search.

Used by: RoofBot, PlumbBot, GardenBot (LA market)

🏘️ Rentcast API

Property data, home valuations, and rental market intelligence. Roof age, property size, and owner contact lookup.

Used by: RoofBot, PlumbBot

🌩️ NOAA / OpenWeather

Storm events, precipitation forecasts, extreme weather alerts. Mapped to zip code for trade-relevant triggers.

Used by: RoofBot, GardenBot, WalkBot

🗺️ Google Maps API

Route optimization, address geocoding, neighborhood clustering, travel time estimates for field routing.

Used by: WalkBot, GardenBot, PlumbBot

📞 311 Open Data

City service requests — water main breaks, flooding complaints, infrastructure failures. Early signal for plumbing demand.

Used by: PlumbBot

🌱 USDA Plant Database

Hardiness zones, planting calendars, species requirements. Seasonal intelligence for landscape and garden operations.

Used by: GardenBot

🛍️ Faire / FashionGo

Wholesale marketplace APIs. Fabric, trim, and finished goods pricing and availability for independent designers.

Used by: FashionBot

📅 Google Calendar + Stripe

Event scheduling, vendor deadline management, payment tracking, and invoice generation for event and service operators.

Used by: PartyBot, WalkBot, FashionBot
